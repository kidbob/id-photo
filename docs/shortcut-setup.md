# iPhone 快捷指令：证件照抠图换底

适用于 **iOS 16+**，使用系统「从图像中移除背景」，无需联网。

## 目标效果

选一张照片 → 自动抠图 → 选白/蓝/红底 → 按一寸或二寸像素导出 → **PNG 存入相簿**（不压 50KB，保留清晰度）。

## 创建步骤

### 1. 新建快捷指令

1. 打开 **快捷指令** App
2. 右上角 **+** → 命名：`证件照抠图`
3. 打开 **在共享表单中显示**（可选，便于从相册分享触发）

### 2. 选取与抠图

| 顺序 | 动作 | 设置 |
|------|------|------|
| 1 | **选取照片** | 关闭「选择多张」；关闭「显示图像编辑器」 |
| 2 | **从图像中移除背景** | 输入为上一步的「照片」 |
| 3 | **如果** | 「抠图结果」**有任意值** |
| 4 | （否则） | **显示提醒**：「抠图失败，请换一张人像更突出的照片」→ **停止快捷指令** |
| 5 | **设定变量** | 名称 `人像`；值为抠图结果 |

### 3. 选择底色

| 顺序 | 动作 | 设置 |
|------|------|------|
| 6 | **从菜单中选取** | 提示「选择底色」；选项：`白色`、`蓝色`、`红色`、`浅蓝` |
| 7 | **如果** | 选取结果 **是** `白色` → **设定变量** `底色` = `#FFFFFF` |
| 8 | **如果** | 选取结果 **是** `蓝色` → **设定变量** `底色` = `#438EDB` |
| 9 | **如果** | 选取结果 **是** `红色` → **设定变量** `底色` = `#FF0000` |
| 10 | **如果** | 选取结果 **是** `浅蓝` → **设定变量** `底色` = `#00BFF3` |

> 色值与 PWA 内 [`src/data/specs.ts`](../src/data/specs.ts) 中 `BG_COLORS` 一致。

### 4. 选择规格并缩放

| 顺序 | 动作 | 设置 |
|------|------|------|
| 11 | **从菜单中选取** | 提示「选择尺寸」；选项：`一寸`、`二寸`、`美国签证` |
| 12 | **如果** 一寸 | **调整图像大小**：宽 **295**、高 **413** 像素；方式 **更精确**；**关闭** 按比例 |
| 13 | **如果** 二寸 | **调整图像大小**：宽 **413**、高 **579** 像素 |
| 14 | **如果** 美国签证 | **调整图像大小**：宽 **600**、高 **600** 像素 |

将缩放后的结果 **设定变量** `缩放人像`。

### 5. 铺底色并合成

快捷指令没有直接的「纯色矩形」动作，用 **生成图像** 或 **网页 HTML 转图像** 较麻烦。推荐简化流程：

**简化版（推荐）**

1. 缩放后变量 `缩放人像` 已是透明底 PNG
2. **存储到相簿**（格式 **PNG**）
3. 打开 **证件照 PWA** → 选该 PNG → 选底色与规格 → **上传模式** 压缩

PWA 会自动检测透明通道并铺底色，无需在快捷指令里叠放。

**完整版（坚持在快捷指令内换底）**

1. **调整图像大小** 对变量 `底色` 不适用；需用 **Markup** 或第三方「纯色图片」动作
2. 或使用 **叠放图像**：底层为生成的纯色图（宽高手动填 295×413），上层为 `缩放人像`，位置居中
3. 输出后再 **存储到相簿** PNG

> iOS 版本不同，动作名称可能为「叠放文本」旁的 **叠放图像**。若找不到，用简化版即可。

### 6. 完成

- 添加 **存储到相簿**
- 运行测试：选一张半身照 → 检查边缘与尺寸
- 快捷指令详情 → **添加到主屏幕**（可选）

## 与 PWA 配合

```
快捷指令：原图 → 抠图 → 按像素缩放 → PNG 相簿
     ↓
证件照 PWA：选 PNG → 铺底色（若透明）→ 压到 ≤50KB → 保存 JPEG
```

## 常见问题

**找不到「从图像中移除背景」**  
需 iOS 16+，在动作库搜索「背景」或「移除」。

**抠图后头发边缘有锯齿**  
换光线更均匀的照片；避免与背景同色头发。

**快捷指令无法精确压到 50KB**  
不要在快捷指令里转 JPEG；交给 PWA 上传模式。
