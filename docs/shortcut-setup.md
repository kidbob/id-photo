# iPhone 快捷指令：证件照抠图（零基础教程）

适用于 **iOS 16 及以上**。用苹果本机 AI 抠图，**比浏览器快、边缘更好**，无需 VPN。

---

## 方式一：一键添加（推荐）

### 第 0 步：打开权限

1. 打开 iPhone **设置**
2. 下滑找到 **快捷指令**，点进去
3. 打开 **允许不受信任的快捷指令**（开关变绿）

> 若看不到这项：先完成下面「方式二」手动建一条并运行一次，再回到设置里看。

### 第 1 步：用 Safari 打开证件照应用

1. 用 **Safari**（不要用微信内置浏览器）打开你的证件照网址  
   例如 Vercel 部署地址，或 `http://电脑IP:5174`
2. 底部点 **说明**
3. 点蓝色大按钮 **「添加快捷指令：证件照抠图」**
4. 自动跳转到「快捷指令」App
5. 点 **添加快捷指令** → **完成**

### 第 2 步：试运行

1. 打开 **快捷指令** App
2. 点 **证件照抠图**
3. 在相册选一张半身照
4. 等待几秒（本机抠图）
5. 弹出「抠图完成」→ 点好
6. 打开 **照片** App，最后一张应为 **透明底 PNG**

### 第 3 步：在证件照 App 里压 KB

1. 打开证件照 PWA（主屏幕图标或 Safari）
2. **制作** 页 → 确认 **「浏览器内自动抠图」已关闭**
3. **选择照片** → 选刚保存的 PNG
4. 选规格（一寸等）、底色（白/蓝/红）
5. **上传模式** → **处理并预览** → 长按图片 **存储到照片**

## 方式二：手动创建（链接导入失败时）

下面每一步都写在「快捷指令」App 里要点哪里。

### 1. 新建快捷指令

| 步骤 | 操作 |
|------|------|
| 1 | 打开 **快捷指令** App（蓝色图标） |
| 2 | 底部点 **快捷指令** 标签 |
| 3 | 右上角 **＋**（加号） |
| 4 | 顶部名称改为 **证件照抠图** |

### 2. 添加「选取照片」

| 步骤 | 操作 |
|------|------|
| 1 | 点 **添加操作**（或中间「搜索操作」） |
| 2 | 搜索 **照片**，选 **选取照片** |
| 3 | 点「选取照片」方块上的 **＞** 展开 |
| 4 | **选择多张** → 关闭 |
| 5 | **显示图像编辑器** → 关闭（若有） |

### 3. 添加「移除图像背景」

| 步骤 | 操作 |
|------|------|
| 1 | 再点 **添加操作** |
| 2 | 搜索 **背景** 或 **移除** |
| 3 | 选 **移除图像背景**（图标像抠图的人像） |
| 4 | 点该方块 **＞** 展开 |
| 5 | **输入** 点一下 → 选变量 **照片**（上一步输出） |

> iOS 16 以下没有此动作，需升级系统。

### 4. 添加「存储到相簿」

| 步骤 | 操作 |
|------|------|
| 1 | 添加操作，搜索 **存储** |
| 2 | 选 **存储到相簿** |
| 3 | **输入** → 选 **图像**（移除背景那一步的输出） |
| 4 | 若有 **格式**，选 **PNG**（保留透明底） |

### 5. 添加完成提示（可选）

| 步骤 | 操作 |
|------|------|
| 1 | 添加操作，搜索 **显示提醒** |
| 2 | 标题：**抠图完成** |
| 3 | 正文：**请打开证件照 App，关闭自动抠图，选择该照片后压缩。** |

### 6. 保存

1. 右上角 **完成**
2. 回到快捷指令列表，应能看到 **证件照抠图**

### 7. 加到主屏幕（可选）

1. 在列表中点 **证件照抠图** 右上角 **⋯**
2. 选 **详细信息**（或点进详情页）
3. **添加到主屏幕**

## 进阶：在快捷指令里顺便缩放到一寸（可选）

若希望抠图后直接是 295×413 像素，在「存储到相簿」**之前**插入：

1. 添加 **调整图像大小**
2. **输入** → 选移除背景后的 **图像**
3. 展开设置：
   - 宽度 **295** 像素
   - 高度 **413** 像素
   - 关闭 **保持比例**（或选「更精确」）
4. 再 **存储到相簿**

二寸：宽 **413**、高 **579**。  
美国签证：宽 **600**、高 **600**。

换底色仍在 **证件照 App** 完成（透明 PNG 铺色更干净）。

## 常见问题

**Q：点「添加快捷指令」没反应？**  
A：必须用 **Safari**；检查设置里「允许不受信任的快捷指令」；链接需在 **HTTPS** 或本地调试地址可访问。

**Q：提示无法验证 / 无法导入？**  
A：用本文 **方式二** 手动创建，效果相同。

**Q：移除背景后没有图？**  
A：原图背景与人像对比度太低，换纯色背景、光线均匀的照片。

**Q：快捷指令能压到 50KB 吗？**  
A：很难精确。请交给证件照 App 的 **上传模式**。

**Q：能和网上一样分享 iCloud 链接吗？**  
A：可以。手动创建成功后，在快捷指令详情里点 **分享** → 复制 iCloud 链接，发给自己备用。

## 一键链接原理（给开发者）

Safari 打开：

```
shortcuts://import-shortcut?url=快捷指令文件HTTPS地址&name=证件照抠图
```

本项目构建时生成 `public/shortcuts/zhengjianzhao-koutu.shortcut`，部署后与记账一样托管在 Vercel 静态资源上。
